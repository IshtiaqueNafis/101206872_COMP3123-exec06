# Week06 Exercise - Notes Application

Build a Restful CRUD API for a simple Note-Taking application using Node.js, Express and MongoDB.

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Server

```bash
node server.js
```

3. Seed data  
```seed
node .\seeder.js -i
```

3. Delete data
```delete
node .\seeder.js -d
```

You can browse the apis at <http://localhost:8081>
